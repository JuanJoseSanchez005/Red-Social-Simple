# 📱 Red Social Simplificada

Este proyecto es una práctica de desarrollo web que simula una **red social básica**.  
Incluye un sistema de inicio de sesión, publicaciones recientes con botones de "Me gusta" y una galería de imágenes interactiva.

---

## 🚀 Características
- **Login simple** con validación de usuario y contraseña.
- **Página de inicio (home.html)** con publicaciones dinámicas y galería de imágenes.
- **Perfil de usuario (profile.html)** con tarjeta personalizada e inspiradora.
- **Galería interactiva**: los botones alternan entre "❤️ Me gusta" y "💙 Te gusta".
- **Diseño responsivo** gracias a Bootstrap.
- **Estilos personalizados** en `styles.css`.

---

## 📂 Estructura del Proyecto
├── index.html        # Página de inicio de sesión
├── home.html         # Página principal con publicaciones y galería
├── profile.html      # Perfil del usuario
├── gallery-mockup.html # Versión de prueba de la galería
├── styles.css        # Estilos personalizados
├── script.js         # Lógica en JavaScript
└── assets/           # Carpeta de imágenes y recursos

---

## 🛠️ Tecnologías Utilizadas
- **HTML5** para la estructura.
- **CSS3** y **Bootstrap 5** para estilos y diseño responsivo.
- **JavaScript** para la lógica de login, publicaciones y galería.
